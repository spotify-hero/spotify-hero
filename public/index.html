Spotify Web Playback SDK Quick Start Tutorial

1. Vérifier le message dans la console : Ready with Device ID 8809298a4f83da63f1e75c52d6a8c7df0c5f35eb
2. Aller dans l'onglet Spotify Spotify > Se connecter à un appareil > Web SDK
3. A partir de là, on peut fermer l'onglet Spotify et contrôler la lecture avec des requêtes cURL

Si la musique n'est pas lue dans l'onglet, essayer de désactiver AdBlock, s'assurer que les DRM sont activées.
Si ça ne marche toujours pas, essayer avec Firefox
